# Tech Stack
| Layer | Technology |
|-------|------------|
| Frontend | Next.js 16, React, TailwindCSS, Mobile-first |
| Backend | AWS Lambda Powertools on AWS Lambda + REST API using API Gateway |
| AI/ML | AWS Bedrock (Claude Sonnet 4.5 and/or Claude Haiku 4.5), Pydantic AI |
| Database | DynamoDB (single-table or table-per-entity) |
| Auth | Cognito (lightweight for demo) |
| Banking | Plaid, Capital One Nessie and/or VISA APIs (sandbox mode) |
| Product Data | ASOS Affiliate API + mock fallback |
| Browser Extension | Chrome Manifest V3 (stretch goal) |
| Deployment | Amplify (frontend), SAM/CDK (backend) |
| Code Review | CodeRabbit (sponsor prize!) |
